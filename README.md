# 📊 West Bengal Election 2026 – Data Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing the **West Bengal Election 2026 dataset** using Microsoft Excel. The goal is to uncover meaningful insights related to **vote share, voter turnout, party performance, and regional trends** through an interactive dashboard built with pivot tables and charts.

The dashboard enables users to explore election data dynamically and understand patterns across **districts, phases, gender, and urban vs rural regions**.

## ❗ Problem Statement
Election data is complex and multi-dimensional, making it difficult to analyze directly. This project aims to:

- Clean and structure raw election data  
- Analyze vote share and turnout trends  
- Compare party performance across districts and phases  
- Build a user-friendly dashboard for better decision-making  

---

## ❓ Research Questions
1. Which party has the highest overall vote share?  
2. How does vote share vary across different districts?  
3. What trends are observed across election phases?  
4. Is there a relationship between turnout and vote share?  
5. How do urban and rural voting patterns differ?  
6. Which districts show strong party dominance?  
7. How does gender impact voting trends?  
8. Which districts have high or low turnout?  
9. Which phase has the highest voter participation?  
10. How do parties perform in urban vs rural regions?  
11. What is the relationship between polling booths and total votes?  
12. Are there shifts in voter behavior across phases?  

---

## 📈 Key Insights
- TMC holds a strong lead in overall vote share  
- BJP and INC show competitive performance  
- CPI(M) maintains steady but moderate support  
- Urban regions show competitive voting patterns  
- Rural areas often show strong party dominance  
- Turnout varies across districts, affecting final results  
- Vote share remains relatively stable across phases
